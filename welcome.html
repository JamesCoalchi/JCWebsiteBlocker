Welcome to JC Website Blocker!

Thank you for installing the JC Website Blocker extension!

Use this extension to block websites and automatically close tabs containing those sites.

How to Use JC Website Blocker:

  1. Click on the Extensions button in your browser's toolbar.
  2. Select JC Website Blocker from the dropdown menu.
  3. In the input field, enter the URL of the website you want to block.
    • Ensure you include the protocol (e.g., https:// or http://).
    • Exclude subdomains (e.g., use https://youtube.com instead of https://www.youtube.com).
  4. Click the Add to Block List button to save the URL to the block list.
